01 — 3 LFOs Per Track

Every audio track on the Octatrack has three dedicated LFOs (Low Frequency Oscillators). These are automated modulation sources that continuously change parameter values for you — creating movement, rhythm, and life in your sounds without touching a single knob.

Accessing the LFO Page

Select a track using the [TRACK] buttons ([T1] through [T8]), then press the [LFO] button to open the LFO MAIN page. You will see six parameters:

LFO MAIN Page Parameters

Parameter What It Controls
SPD1 Speed of LFO1 — higher = faster modulation
SPD2 Speed of LFO2
SPD3 Speed of LFO3
DEP1 Depth (amount) of LFO1 modulation
DEP2 Depth of LFO2 modulation
DEP3 Depth of LFO3 modulation

LFOs are always synced to the BPM of the Octatrack. For LFO speeds that lock to straight beats, try SPD values of 16, 32, or 64 .

PRO TIP: Beat-Synced Speeds

The MULT parameter in the [LFO] SETUP page (press [FUNC] + [LFO]) multiplies the base SPD value. Combined, SPD and MULT let you range from glacially slow sweeps to audio-rate modulation. The key musical values:

  • SPD 16 = one full cycle per bar (4/4 time)
  • SPD 32 = two cycles per bar (8th note feel)
  • SPD 64 = four cycles per bar (16th note feel)
  • SPD 128 = eight cycles per bar (32nd note feel)

LFO Routing Diagram

IMPORTANT: MAIN Pages Only

LFOs can only modulate parameters found on MAIN pages (accessed by pressing [SRC], [AMP], [FX1], [FX2], [LFO], or [MIX] once). Parameters on SETUP pages (accessed via [FUNC] + [TRACK PARAMETER]) cannot be LFO targets. They also cannot be parameter-locked or assigned to scenes.

02 — LFO Modes: FREE, TRIG, HOLD, ONE & More

Each LFO has a TRIG mode setting that determines when and how the LFO cycles. Access this by pressing [FUNC] + [LFO] to open the LFO SETUP page, then navigate to LFO 1, 2, or 3 using the [UP]/[DOWN] arrow keys.

Mode Behavior Best For
FREE Runs continuously, never restarts Evolving textures, slow drifting pads, organic movement
TRIG Restarts from zero on every sample trig, then runs until next trig Wobble bass, per-hit filter sweeps, synchronized effects
HOLD Runs in the background but latches its value on each trig Random-ish stepped modulation, sample & hold textures
ONE Restarts on trig, runs one full cycle, then stops Envelope-like behavior: filter plucks, pitch drops, volume swells
HALF Restarts on trig, runs half a cycle, then stops Attack-only envelopes, one-way sweeps
SYNC TRIG Restarts on track start and every pattern loop Pattern-length sweeps that always reset with the loop
SYNC ONE Restarts on track start, plays one period One-shot pattern-level envelopes
SYNC HALF Restarts on track start, plays half a period Build-up ramps across a pattern

PRO TIP: TRIG Mode for Wobble Bass

The classic dubstep wobble: set LFO1 to TRIG mode with a triangle wave targeting the filter cutoff in [FX1]. Every time a step triggers, the LFO restarts — giving you perfectly synchronized filter sweeps. Adjust SPD to control the wobble rate.

03 — LFO Designer

The LFO Designer lets you draw custom waveforms using the 16 [TRIG 1] through [TRIG 16] keys as faders. Each trig key represents one step of the waveform. This goes far beyond simple sine and triangle shapes — you can create rhythmic gate patterns, complex modulation shapes, and pseudo-sequencer effects.

How to Access

1

Open LFO Setup

Press [FUNC] + [LFO] to open the LFO SETUP page.

2

Navigate to Designer

Use the [UP]/[DOWN] arrow keys to navigate past LFO 1, 2, and 3 to the DESIGNER page at the bottom of the left column.

3

Draw Your Waveform

Hold a [TRIG 1] through [TRIG 16] key and turn a DATA ENTRY knob (top row) to set that step's value. The waveform updates live on screen.

4

Interpolation (Optional)

Hold a [TRIG] key and press [TEMPO] to add smooth interpolation between that step and the next. Interpolated steps show as yellow [TRACK] keys.

LFO Designer Quick Reference

  • 16 steps per waveform — each step has a unique value
  • Stepped or interpolated — toggle per-step with [TRIG] + [TEMPO]
  • 8 designer slots per track (labeled T1-T8) — all available to all 3 LFOs
  • Randomize: [FUNC] + [BANK] then select RANDOMIZE
  • Invert: [FUNC] + [BANK] then select INVERT
  • Rotate: [FUNC] + [LEFT]/[RIGHT] to shift the waveform
  • Copy/Paste: Standard copy operations work — copy waveforms between tracks

Interactive LFO Designer

Draw a Custom Waveform

Click and drag on the bars below to shape your LFO waveform. Each bar represents one of the 16 [TRIG 1]-[TRIG 16] steps.

PRO TIP: Rhythmic Gates

Use the LFO Designer targeting VOL on the [AMP] page with TRIG mode. Set alternating high/low steps to create rhythmic gate patterns. Combine with the [CROSSFADER] and scenes for controllable gating intensity.

04 — Assigning LFO Targets

The real power of LFOs comes from what they modulate . Each LFO targets one parameter, chosen via the PMTR (parameter) setting on the [LFO] SETUP page.

How to Set a Target

1

Open LFO Setup

Press [FUNC] + [LFO] (or double-tap [LFO]) to open the LFO SETUP menu.

2

Select Which LFO

Use [UP]/[DOWN] to navigate to LFO 1, LFO 2, or LFO 3.

3

Set PMTR (Destination)

Turn the [LEVEL] encoder to browse through all available MAIN page parameters. The screen shows which [TRACK PARAMETER] page the target belongs to.

4

Set WAVE, MULT, TRIG, SPD, DEP

Configure the waveform shape, speed multiplier, trig mode, base speed, and modulation depth.

Common LFO Targets

Page Parameter Effect When Modulated
[SRC] PTCH (Pitch) Vibrato, pitch drift, FM-style effects at high speeds
[SRC] STRT (Start) Scanning through sample, granular-like textures (Flex only)
[AMP] VOL (Volume) Tremolo, rhythmic gating, amplitude envelopes
[AMP] PAN Auto-pan, stereo movement, spatial effects
[FX1]/[FX2] Filter Cutoff Wobble, filter sweeps, acid-style squelch
[FX1]/[FX2] Delay Time Pitch-warping echoes, tape-speed effects
[FX1]/[FX2] Delay Send Rhythmic delay throws, dub-style echo bursts
[MIX] MAIN Level Master volume modulation, ducking effects
[LFO] LFO1/2 SPD Meta-modulation: speeding up/slowing down another LFO
[LFO] LFO1/2 DEP Meta-modulation: waxing/waning another LFO's intensity

STATIC MACHINE LIMITATION

The track LFOs cannot modulate the STRT parameter of a Static machine. The crossfader can, but if modulation is too fast the Static machine will not be able to update the start position correctly. Use Flex machines for LFO-driven start-point modulation and granular effects.

SETUP Parameters: 6 Per LFO

Each LFO in the SETUP page has these parameters:

  • PMTR — target parameter (destination)
  • WAVE — waveform shape (11 built-in + 8 designer)
  • MULT — speed multiplier
  • TRIG — trigger mode (FREE, TRIG, HOLD, ONE, HALF, SYNC...)
  • SPD — base speed (synced to BPM)
  • DEP — modulation depth (0-127)

05 — Meta-Modulation

This is where things get deep. LFO3 can modulate LFO1 or LFO2's speed or depth. An LFO modulating another LFO. The result: complex, evolving modulations that would be impossible to create manually.

The Meta-Modulation Rule

An LFO can only modulate LFOs with a lower number :

  • LFO3 can modulate LFO1 SPD, LFO1 DEP, LFO2 SPD, LFO2 DEP
  • LFO2 can modulate LFO1 SPD, LFO1 DEP
  • LFO1 cannot modulate other LFOs (it is the lowest number)

Setting Up Meta-Modulation

1

Configure LFO1 First

In [LFO] SETUP (press [FUNC] + [LFO]), set LFO1 to target your desired parameter — for example, filter cutoff on [FX1]. Set the WAVE to triangle, TRIG mode to TRIG, SPD to 32, DEP to 64.

2

Set LFO3 to Target LFO1 SPD

Navigate to LFO3 in the SETUP page (use [DOWN]). Set PMTR to "LFO SPD1". Now LFO3 will modulate LFO1's speed. Set LFO3 WAVE to sine, TRIG to FREE, SPD to 8, DEP to 32.

3

Listen to the Result

Press [PLAY]. LFO1 sweeps the filter, but its speed now rises and falls according to LFO3's slow sine wave. The wobble speeds up and slows down in a complex, evolving cycle.

Meta-Modulation Recipes

Recipe LFO1 Target LFO3 Target Result
Breathing Wobble [FX1] Cutoff LFO1 SPD Filter wobble that speeds up and slows down rhythmically
Fading Vibrato [SRC] PTCH LFO1 DEP Vibrato that waxes and wanes in intensity
Evolving Pan [AMP] PAN LFO1 SPD Auto-pan that varies in speed — organic stereo movement
Chaos Engine [FX1] Cutoff LFO1 DEP Filter intensity that surges and retreats unpredictably

PRO TIP: LFO2 as a Second Modulator

Do not forget LFO2 can also meta-modulate LFO1. Use LFO1 on filter cutoff, LFO2 on LFO1's speed, and LFO3 on LFO2's depth. Three layers of modulation creates genuinely unpredictable, generative textures — great for ambient and experimental work.

06 — Audio-Rate Tricks

When you push LFO speeds to extreme values (SPD at high settings combined with high MULT multipliers), the LFO enters audio-rate territory . The modulation is so fast that it stops sounding like movement and starts creating new timbres.

What Happens at Audio Rate

Target Effect Sound
[SRC] PTCH FM Synthesis Metallic, bell-like, inharmonic tones — similar to FM synths
[AMP] VOL Ring Modulation Ring-mod-like sidebands, robotic timbres
[FX1] Cutoff Aggressive Filtering Harsh, buzzy filter resonance effects
[SRC] STRT Granular Chaos Rapid sample-scanning creating glitchy, granular textures (Flex only)

How to Set Up Audio-Rate LFO

1. Open [LFO] SETUP: [FUNC] + [LFO] 2. Set PMTR to [SRC] PTCH for FM synthesis effects 3. Set WAVE to SINE (smoothest FM tones) 4. Set MULT to x16 or higher 5. Set SPD to 64-128 6. Set DEP to taste — start low (10-20), increase gradually 7. Press [PLAY] and sweep DEP to control intensity

WARNING: Ear Safety

Audio-rate LFO modulation can produce very loud, harsh, and unexpected sounds . Always start with low DEP values and increase gradually. Keep your monitor volume at a safe level when experimenting with these techniques.

PRO TIP: P-Lock the Depth

Use parameter locks on individual steps to control where the audio-rate effect kicks in. Hold [TRIG 5] and turn the DEP1 knob on the [LFO] page to p-lock the LFO depth only on that step. This way, most of your pattern plays clean while specific hits get the aggressive FM treatment.

07 — Practical Exercises

Put it all together with these hands-on exercises. Load any sample onto a Flex machine on [T1] and follow along.

Exercise 1: Wobble Bass

1

Set Up the Filter

Select [T1], press [FX1], and assign a 2-Pole Lowpass Filter. Set the base cutoff to about 40.

2

Assign LFO1 to Cutoff

Press [FUNC] + [LFO]. Navigate to LFO1. Set PMTR to [FX1] cutoff. Set WAVE to triangle, TRIG mode to TRIG, SPD to 32, DEP to 64.

3

Place Trigs and Play

Press [REC], then press [TRIG 1], [TRIG 5], [TRIG 9], [TRIG 13] for a 4-on-the-floor pattern. Press [PLAY]. You should hear a classic wobble bass with the filter opening and closing on every beat.

4

Experiment

Change SPD on the [LFO] MAIN page (press [LFO]) to alter the wobble rate. Try SPD 16 for slower, SPD 64 for faster. P-lock different SPD values per step for varying wobble speeds.

Exercise 2: Auto-Pan

1

Assign LFO2 to PAN

Press [FUNC] + [LFO], navigate to LFO2. Set PMTR to [AMP] PAN. Set WAVE to sine, TRIG mode to FREE, SPD to 8, DEP to 50.

2

Listen

Press [PLAY]. The sound drifts slowly between left and right speakers. Because FREE mode never restarts, the panning is continuous and organic.

Exercise 3: Evolving Meta-Mod Texture

1

LFO1: Filter Sweep

Set LFO1 PMTR to [FX1] cutoff. WAVE = saw, TRIG = TRIG, SPD = 32, DEP = 64.

2

LFO3: Modulate LFO1 Speed

Set LFO3 PMTR to LFO SPD1. WAVE = sine, TRIG = FREE, SPD = 4, DEP = 40. Now LFO1's filter sweep will accelerate and decelerate over time.

3

Let It Evolve

Press [PLAY] and listen for 30+ seconds. The sound constantly changes as LFO3 pushes LFO1 through different speed ranges. No two bars sound exactly the same.

Exercise 4: LFO Designer Rhythmic Gate

1

Open LFO Designer

Press [FUNC] + [LFO], navigate to DESIGNER using [DOWN]. Draw a pattern: set [TRIG 1], [TRIG 3], [TRIG 5], [TRIG 9], [TRIG 11] to maximum (127) and all others to 0. This creates an irregular rhythmic pattern.

2

Assign to VOL

Navigate to LFO1 in SETUP. Set PMTR to [AMP] VOL. Set WAVE to the designer waveform (T1). Set TRIG to TRIG, SPD to 64, DEP to 127.

3

Play and Adjust

Press [PLAY]. The volume gates rhythmically according to your custom pattern. Go back to the DESIGNER and reshape the waveform while it plays — changes are instant.
