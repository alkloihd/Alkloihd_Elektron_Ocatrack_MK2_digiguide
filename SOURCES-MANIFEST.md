## Attribution Approach

### Fair Use Compliance

All content described in the ecosystem follows these principles:

1. **Respect Original Creators**
   - Every video, guide, resource is credited with creator name and link
   - No re-hosting of third-party content
   - Links drive traffic to original creators

2. **Copyright Acknowledgment**
   - Official Elektron resources cited with page numbers
   - Merlin's community guide referenced with Elektronauts link
   - YouTube videos embedded (not re-hosted)
   - Forum posts linked (not reproduced)

3. **Transparency**
   - Sources clearly listed
   - Creator roles and specialties documented
   - Platform links provided for discovery

### Attribution Standards

| Source Type         | Citation Method       | Link?            |
| ------------------- | --------------------- | ---------------- |
| Official Manual     | Page number reference | Link to PDF      |
| Community Guide     | Section + author      | Link to thread   |
| YouTube Videos      | Creator name + title  | Embed iframe     |
| Forum Posts         | Author + thread link  | Direct link      |
| Educational Courses | Course name + creator | Direct to course |

## Sources Not Included (And Why)

### Content Excluded

1. **Paid Courses (Paywalled)**
   - Referenced for awareness, not embedded
   - Link to course page instead
   - Respect creator monetization

2. **Outdated MK1-Only Content**
   - Included only if foundational concepts apply to MK2
   - Noted as "MK1-specific" when included

3. **Live Performances (No Teaching Intent)**
   - Included only if educational commentary provided
   - Links to performances available via creators

4. **Unverified User Content**
   - Community tips only after curator approval
   - Opt-in sharing with attribution
