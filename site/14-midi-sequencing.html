MIDI Tracks Overview

The Octatrack MK2 is not just an audio sampler — it contains a full 8-track MIDI sequencer that runs alongside the 8 audio tracks. These MIDI tracks do not generate any sound internally. Instead, they send note data, CC messages, program changes, and pitch bend information out through the MIDI OUT port to control external synthesizers, drum machines, and other MIDI-equipped gear.

Press the [MIDI] button to enter MIDI mode. The [MIDI] LED lights up and the screen switches to show the 8 MIDI tracks instead of the 8 audio tracks. Each MIDI track has its own:

  • 16-step sequencer with the same trig system as audio tracks
  • Parameter locks on every step (note, velocity, CC values)
  • 3 dedicated MIDI LFOs for automating external parameters
  • Built-in arpeggiator with multiple modes and up to 16-step custom patterns
  • 10 assignable CC slots across two CTRL pages
  • Micro timing and trig conditions (FILL, probability, A:B — same as audio)

Accessing MIDI Mode

1

Press [MIDI] to switch to MIDI mode. The LED lights up.

[MIDI]
2

Select a MIDI track by pressing [T1] through [T8]. The screen shows the MIDI channel assigned to each track. An "X" means no MIDI channel is assigned.

[T1] through [T8]
3

Press [MIDI] again to return to audio mode.

Manual reference: Section 15 — The MIDI Sequencer (page 90). Press [FUNC] + [MIDI] to jump directly to the MIDI SYNC settings.

Channel conflict warning: If an audio track and a MIDI track share the same MIDI channel, the MIDI track will block the audio track from sending out data, and the audio track will block the MIDI track from receiving data. Avoid assigning the same MIDI channels to both audio and MIDI tracks.

MIDI Tracks vs Audio Tracks

Feature Audio Tracks MIDI Tracks
Sound generation Yes (samples, inputs) No (data only)
16-step sequencer Yes Yes
Parameter locks Yes Yes
Trig conditions Yes Yes
Micro timing Yes Yes
Slide trigs Yes No
Recorder trigs Yes No
LFOs 3 per track 3 per track (MIDI-specific)
Arpeggiator No Yes (built-in)
Scene locks Yes No

Muting tip: When in mute mode ([FUNC] + [BANK] on MK2), [TRIG 1] through [TRIG 8] mute audio tracks and [TRIG 9] through [TRIG 16] mute MIDI tracks. This works regardless of whether you are in audio or MIDI editing mode.

Note Modes

Each MIDI track has a NOTE MAIN page and a NOTE SETUP page. These are accessed by pressing [SRC] for the main page, or [FUNC] + [SRC] (or double-tap [SRC]) for the setup page, while in MIDI mode.

NOTE MAIN Page Parameters

The NOTE MAIN page is where you set what notes the MIDI track sends out. Access it by pressing [SRC] while in MIDI mode.

Parameter Knob What It Does
NOTE A Root note to send. Hold a [TRIG] key in [REC] mode to see a mini-keyboard on screen. Turn knob A to set the note.
VEL B Velocity of the note (0–127). A value of 0 sends a NOTE OFF command.
LEN C Duration of the note. When it expires, a NOTE OFF is sent. Max = infinite length. Also controls arpeggio duration.
NOT2–NOT4 D–F Add up to 3 additional notes offset from the root to create chords (up to 4 notes total). Value of 0 removes the offset note.

Chord trick: Set NOTE to C3, NOT2 to +4 (E3), NOT3 to +7 (G3), NOT4 to +12 (C4) — you now have a C major chord with an octave doubling, all from one trig on one MIDI track.

NOTE SETUP Page Parameters

Access by pressing [FUNC] + [SRC] or double-tapping [SRC] while in MIDI mode. This is where you assign the MIDI channel.

Parameter Knob What It Does
CHAN A MIDI channel this track sends data on (1–16, or OFF). Press the knob to confirm your selection.
BANK B Bank change message (MSB, 0–127). Sent when switching to a pattern linked to another Part. Max = OFF.
PROG C Program change message (0–127). Sent when switching to a pattern linked to another Part. Max = OFF.
SBNK D Sub-bank change message (LSB, 0–127). Max = OFF.

Important: When you select a value in the NOTE SETUP menu, the parameter graphic becomes inverted. This means the value has not been activated yet. You must press the corresponding DATA ENTRY knob or [YES] to confirm and send the change.

Trig Modes in MIDI Mode

The [TRIG] keys behave differently in MIDI mode depending on the current trig mode:

  • STANDARD: [TRIG 1]–[TRIG 8] send notes C1–G1, [TRIG 9]–[TRIG 16] send C2–G2
  • CHROMATIC: [TRIG] keys send note on/off commands on the active MIDI track's channel, mapped chromatically across 2+ octaves
  • SLOTS / SLICES: [TRIG] keys map to sound slots or slice positions — useful for controlling external samplers

Manual reference: Section 15.4.1 — NOTE MAIN (page 91), Section 15.4.2 — NOTE SETUP (page 92).

MIDI Arpeggiator

Every MIDI track has a built-in arpeggiator that generates rhythmical note progressions from the notes you place on each trig. The arpeggiator takes your root note (and any chord notes from NOT2–NOT4) and plays them in sequence at a speed synchronized to the project BPM.

Arpeggiator MAIN Page

Access by pressing [AMP] while in MIDI mode. In MIDI mode, the [AMP] key opens the ARPEGGIATOR parameter pages instead of the AMP pages used by audio tracks.

Parameter Knob What It Does
TRAN A Transpose the arpeggio up or down by semitones. Affects note trigs even if MODE is OFF.
LEG B Legato mode. ON = overlapping notes play legato (old note released after new one). OFF = note off sent before each new note.
MODE C Arpeggiator mode — see table below
SPD D Speed, synced to BPM. 6 = 16th notes, 12 = 8th notes, 24 = quarter notes.
RNGE E Octave range. After each arpeggio cycle, notes transpose up one octave until reaching this limit, then reset.
NLEN F Note length of arpeggiated notes (separate from the LEN parameter on NOTE MAIN).

Arpeggiator Modes

Mode Behavior
OFF Arpeggiator disabled
TRUE Plays notes in the order they were inserted
UP Ascending order, lowest to highest, per octave
DOWN Descending order, highest to lowest, per octave
CYCL Ascending then descending (ping-pong)
SHFL Random order within each octave range; moves to next octave after all notes played
RANDOM Fully random playback of arpeggiated notes

Arpeggiator SETUP Page

Access by pressing [FUNC] + [AMP] or double-tapping [AMP] while in MIDI mode. This page lets you customize the arpeggio pattern step by step.

  • The [TRIG] keys show the arpeggio length — lit keys are active steps, set length with knob C
  • Green [TRIG] = active arpeggio step. Press to deactivate (turns red ).
  • Hold a [TRIG] key and turn the [LEVEL] knob or press [UP]/[DOWN] to add a note offset to that step
  • Knob F sets a key scale constraint — when not OFF, all notes are quantized to the selected scale
  • [FUNC] + [LEFT]/[RIGHT] rotates the arpeggio sequence

Pro tip: Set a 4-note chord (NOTE + NOT2–NOT4), then set MODE to UP with RNGE = 2OCT and SPD = 6. You get a classic 16th-note arpeggio spanning 2 octaves — perfect for acid-style basslines or trance leads on an external synth. P-lock the MODE on different steps to switch between UP and DOWN mid-pattern.

Manual reference: Section 15.4.3 — Arpeggiator Main (page 92), Section 15.4.4 — Arpeggiator Setup (page 93).

MIDI LFO & CC Control

Each MIDI track has 3 dedicated MIDI LFOs and 10 assignable CC slots split across two CTRL pages. This is how you automate parameters on external synths — filter sweeps, resonance movements, effect sends, and anything else your gear exposes via MIDI CC.

CTRL 1 Page (4 CC Slots + Pitch Bend + Aftertouch)

Access the CTRL 1 MAIN page by pressing [FX1] while in MIDI mode. Access CTRL 1 SETUP by pressing [FUNC] + [FX1] or double-tapping [FX1].

Page Parameters What They Do
CTRL 1 MAIN PB , AT , CC1–CC4 PB = pitch bend, AT = aftertouch, CC1–CC4 = values for the CCs assigned in SETUP. Default is OFF — enable with [FUNC] + knob press.
CTRL 1 SETUP CC1–CC4 Select which CC numbers CC1–CC4 control. Use [FUNC] + knob C–F for MIDI CC LEARN .

CTRL 2 Page (6 CC Slots)

Access the CTRL 2 MAIN page by pressing [FX2] while in MIDI mode. Access CTRL 2 SETUP by pressing [FUNC] + [FX2] or double-tapping [FX2].

Page Parameters What They Do
CTRL 2 MAIN CC5–CC10 Values for the CCs assigned in CTRL 2 SETUP. Same enable/disable as CTRL 1.
CTRL 2 SETUP CC5–CC10 Select which CC numbers CC5–CC10 control. MIDI CC LEARN available here too.

MIDI CC Learn Workflow

Instead of looking up CC numbers in your synth's manual, use MIDI CC LEARN:

1

In MIDI mode, select the track controlling your synth.

[MIDI] → [T1]
2

Open CTRL 1 SETUP by double-tapping [FX1] or pressing [FUNC] + [FX1].

[FUNC] + [FX1]
3

Press [FUNC] + knob C to enter MIDI CC LEARN for the CC1 slot.

[FUNC] + knob C
4

Turn the cutoff knob on your synth. The OT reads the incoming CC message and auto-assigns it to the CC1 slot.

MIDI LFOs

Access the MIDI LFO MAIN page by pressing [LFO] while in MIDI mode. The 3 MIDI LFOs work exactly like audio LFOs (same modes: FREE, TRIG, HOLD, ONE, HALF, etc.) but they are exclusive to MIDI tracks and can target any MIDI track parameter, including CC values.

The MIDI LFO SETUP page (press [FUNC] + [LFO]) lets you set LFO destinations and access the LFO Designer for custom waveforms. MIDI LFO designer waveforms are separate from audio LFO designer waveforms.

Filter sweep recipe: Assign CC1 to your synth's filter cutoff (via MIDI CC LEARN). Then on the MIDI LFO SETUP page, set LFO1's destination to CC1. Set LFO1 to TRIG mode, triangle wave, SPD = 24 (1 bar cycle), DEPTH = 64. Every note trig will now start a smooth filter sweep on your external synth, all sequenced from the Octatrack.

Manual reference: Section 15.4.5 — MIDI LFO Main (page 93), Section 15.4.6 — MIDI LFO Setup (page 94), Section 15.4.7–15.4.10 — CTRL 1/2 Main & Setup (pages 94–95).

Sync Setup

The Octatrack can be a MIDI master (sending clock and transport to other gear) or a slave (receiving clock from a DAW or another device). All sync settings live under PROJECT > MIDI > SYNC.

Accessing Sync Settings

[FUNC] + [MIDI] or [PROJ] → MIDI → SYNC → [YES]

Sync Parameters

Parameter What It Does
TRANSPORT SEND When active, the OT sends system realtime messages: play, stop, continue, song position pointer.
TRANSPORT RECEIVE When active, the OT responds to incoming play, stop, continue, and song position pointer.
CLOCK SEND When active, the OT transmits MIDI clock. External gear syncs to OT tempo.
CLOCK RECEIVE When active, the OT receives MIDI clock from an external device and syncs its BPM to it.
PROG CH SEND Sends program change messages when the active OT pattern changes. Useful for switching presets on synths when you switch patterns.
CHANNEL (send) Which MIDI channel program changes are sent on. AUTO = first audio track channel not used by any MIDI track.
PROG CH RECEIVE OT listens for incoming program change messages to switch its own patterns externally.
CHANNEL (receive) Which MIDI channel to listen on for incoming program changes. AUTO = any channel in the CHANNELS menu.

Sync timeout: If both CLOCK RECEIVE and TRANSPORT RECEIVE are enabled, the OT will wait up to 16 seconds for MIDI clock after receiving a start/continue message before considering external sync lost. If only TRANSPORT RECEIVE is enabled, it starts immediately after receiving start/continue.

Setting Up OT as Master (Most Common)

1

Connect a MIDI cable from the OT's MIDI OUT to your synth's MIDI IN.

2

Open SYNC settings: press [FUNC] + [MIDI] or navigate [PROJ] > MIDI > SYNC > [YES].

[FUNC] + [MIDI]
3

Enable CLOCK SEND and TRANSPORT SEND by pressing [YES] on each.

4

Make sure your external gear is set to receive external MIDI clock.

5

Press [PLAY] on the OT — all connected gear should start in sync.

[PLAY]

Ableton Integration

The Octatrack and Ableton Live make a powerful combo. You can use the OT as a MIDI master controlling Ableton, or let Ableton be the master with the OT as a slave. Here are the most practical setups.

Connection Options

Method What You Need Pros / Cons
DIN MIDI MIDI interface on your computer + DIN MIDI cables Rock-solid timing, no drivers needed on OT side. Requires a MIDI interface.
USB MIDI USB cable (OT MK2 sends MIDI over USB) Convenient, no extra hardware. May have slightly more jitter than DIN. Cannot use USB Disk Mode at the same time.

Setup A: OT as MIDI Master, Ableton as Slave

Best for: live jams where the OT drives the tempo and Ableton provides additional sounds.

1

Connect OT MIDI OUT (DIN or USB) to your computer.

2

On the OT: [FUNC] + [MIDI] → enable CLOCK SEND and TRANSPORT SEND.

[FUNC] + [MIDI]
3

In Ableton: Preferences > Link/Tempo/MIDI. Under MIDI Ports, find the OT's MIDI input. Enable Sync for that input. Set Ableton to "EXT" sync mode (top-left of the arrangement).

4

Press [PLAY] on the OT. Ableton follows the OT's tempo and starts/stops with it.

[PLAY]

Setup B: Ableton as Master, OT as Slave

Best for: studio production where Ableton is the main DAW and OT provides sampling/effects.

1

Connect MIDI from your computer to OT MIDI IN.

2

On the OT: [FUNC] + [MIDI] → enable CLOCK RECEIVE and TRANSPORT RECEIVE.

[FUNC] + [MIDI]
3

In Ableton: Preferences > Link/Tempo/MIDI. Find the OT's MIDI output. Enable Sync for that output.

4

Press Play in Ableton. The OT locks to Ableton's tempo and starts playback.

Sending MIDI Notes from OT to Ableton Instruments

You can use OT MIDI tracks to sequence soft synths in Ableton:

  1. Press [MIDI] on the OT and select a MIDI track.
  2. Open NOTE SETUP ([FUNC] + [SRC]) and set CHAN to a specific MIDI channel (e.g., channel 1).
  3. In Ableton, create a MIDI track and set its input to the OT's MIDI port, channel 1.
  4. Load any instrument (Wavetable, Operator, a VST) on that Ableton track.
  5. Arm the Ableton track for monitoring. Place note trigs on the OT — Ableton plays the sound.

Pro tip: Use multiple OT MIDI tracks on different channels to sequence multiple Ableton instruments simultaneously. You get the OT's p-locks, trig conditions, and arpeggiator controlling your software synths — the best of both worlds.

USB Disk Mode conflict: You cannot use USB MIDI and USB Disk Mode at the same time. If you need to transfer files to the CF card, you must disconnect MIDI first. Use DIN MIDI if you want to keep the sync connection active while managing files.

Practical Exercise: Control an External Synth

In this exercise you will set up a MIDI track to send notes and CC automation to an external synthesizer (or a software synth via MIDI). By the end, you will have a working pattern with note trigs, velocity p-locks, an arpeggiator, and an LFO-driven filter sweep.

What You Need

  • Octatrack MK2
  • An external synth with MIDI IN (or a computer running a soft synth, connected via MIDI)
  • MIDI cable (DIN or USB)

Step-by-Step

1

Enter MIDI mode

Press [MIDI] to switch to MIDI editing mode. The [MIDI] LED lights up.

[MIDI]
2

Select MIDI track 1

Press [T1] to select the first MIDI track.

[T1]
3

Set the MIDI channel

Open NOTE SETUP by double-tapping [SRC] or pressing [FUNC] + [SRC]. Turn knob A to set CHAN to the MIDI channel your synth listens on (e.g., channel 1). Press knob A or [YES] to confirm.

[FUNC] + [SRC] → CHAN = 1 → [YES]
4

Place note trigs

Press [REC] to enter Grid Recording mode. Press [TRIG 1], [TRIG 5], [TRIG 9], and [TRIG 13] to place trigs on the four beats. Your synth should already be making sound if connected.

[REC] → [TRIG 1] [TRIG 5] [TRIG 9] [TRIG 13]
5

Set different notes per trig (p-locks)

Go to the NOTE MAIN page by pressing [SRC]. Hold [TRIG 1] and turn knob A to set its note (e.g., C3). Hold [TRIG 5] and turn knob A to a different note (e.g., E3). Do the same for [TRIG 9] (G3) and [TRIG 13] (C4). You have just p-locked a melody.

[SRC] → hold [TRIG 1] + turn knob A
6

Add velocity variation

Still on NOTE MAIN, hold [TRIG 9] and turn knob B to lower the velocity (e.g., 80). Hold [TRIG 13] and set velocity to 127 for an accent. This p-locks velocity per step.

hold [TRIG 9] + turn knob B (VEL)
7

Enable the arpeggiator

Press [AMP] to go to the Arpeggiator MAIN page (in MIDI mode, [AMP] opens the arpeggiator). Turn knob C to set MODE to UP. Set SPD (knob D) to 6 for 16th-note arpeggios. Set RNGE (knob E) to 2OCT.

[AMP] → MODE = UP, SPD = 6, RNGE = 2OCT
8

Set up CC control for filter

Open CTRL 1 SETUP: press [FUNC] + [FX1]. Press [FUNC] + knob C to enter MIDI CC LEARN. Turn your synth's filter cutoff knob — the CC number auto-assigns to CC1.

[FUNC] + [FX1] → [FUNC] + knob C → turn synth knob
9

Add LFO to the filter CC

Press [LFO] to open MIDI LFO MAIN. On the LFO SETUP page ([FUNC] + [LFO]), set LFO1's destination to CC1. Back on LFO MAIN, set the waveform to TRI (triangle), SPD to 12, DEPTH to 64. Now the filter sweeps automatically, synced to the pattern tempo.

[LFO] → set waveform, speed, depth
10

Play it!

Press [PLAY] to hear your MIDI-sequenced pattern with arpeggiated notes and automated filter sweeps. Try pressing [YES] to hear any FILL condition trigs you might add later.

[PLAY]
