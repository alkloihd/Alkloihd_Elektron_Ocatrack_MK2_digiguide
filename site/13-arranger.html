Why Arranger?

So far you have been working with one pattern at a time — building beats, adding trig conditions, sculpting scenes. But a single pattern is a loop. To build a full song or live set , you need a way to chain patterns together with transitions, repeats, and tempo changes. That is what the Arranger does.

The Octatrack offers two approaches to multi-pattern playback:

  • Pattern Chaining — A quick way to queue patterns in the same bank. No special mode needed, but limited to same-bank patterns.
  • Arranger (Song Mode) — A full timeline editor with up to 256 rows. Supports patterns from any bank, repeat counts, tempo changes, mutes, scene assignments, and special commands like HALT, LOOP, and JUMP.

Each project can hold 8 separate arrangements . Think of each as a different setlist or song structure.

Entering Arrangement Mode

Press [FUNC] + [ARR] to toggle Arrangement mode on and off. When active, the [ARR] key lights up and the screen changes to show the currently active arrangement.

[FUNC] + [ARR] → [ARR] LED lights up

Press [ARR] alone to open the ARRANGER menu where you can edit, rename, change, chain, clear, save, and reload arrangements.

[ARR] → ARRANGER menu opens

Manual reference: Section 14 — The Arranger (page 87). Section 12.2.3 — Pattern Chaining (page 64).

Pattern Chaining

Pattern chaining is the simplest way to play multiple patterns back-to-back. It requires no special mode — you can do it while a pattern is playing.

How to Chain Patterns

1

Start a pattern playing

Press [PLAY] with any pattern selected. The pattern loops normally.

[PLAY]
2

Hold [PTN] and press trigs in order

Press and hold [PTN], then press and hold [TRIG] keys in the order you want the patterns to play. For example, to chain patterns A01 → A03 → A05:

[PTN] hold + [TRIG 1] hold + [TRIG 3] hold + [TRIG 5]
3

Release and listen

The chained patterns play in sequence. A double arrow on the screen shows the next queued pattern. Once the chain completes, it loops back to the beginning.

Chain Rules

Rule Detail
Same bank only You can only chain patterns within the current bank (e.g., A01 through A16). You cannot chain A03 to B07.
No duplicates A chain cannot contain the same pattern twice. If you need A01 to play twice, use the Arranger with repeats instead.
Quantized to end Pattern changes happen at the end of the current pattern, keeping everything in time.
[STOP] resets Pressing [STOP] once stops playback and resets to the start of the chain. Pressing [STOP] again unlinks all patterns in the chain.

Quick select: To switch to a single pattern without chaining, just press [PTN] + a single [TRIG] key (without holding). The new pattern queues up and starts at the end of the current one. For a different bank, press [BANK] first, select the bank letter, then choose a pattern.

Limitation: If your song needs the same pattern to repeat (like verse-chorus-verse), chaining cannot do this — the same pattern cannot appear twice in a chain. This is one of the key reasons to use the Arranger instead.

Arranger Editor

The Arranger editor is a timeline where each row represents one slot for a pattern (or a special command). This is where you build your song structure.

Opening the Editor

1

Open the Arranger menu

Press [ARR] to open the ARRANGER menu.

[ARR]
2

Select EDIT

Choose EDIT from the menu and press [YES] to open the arrangement editor.

EDIT → [YES]

Editor Columns

Each row in the editor has several columns. Use [LEFT] and [RIGHT] to move between columns, and [UP] and [DOWN] to navigate between rows.

Column What It Does How to Set
ROW Row number (000 to 255). The arrangement plays from row 0 to the end. Navigate with [UP] / [DOWN]
PAT Which pattern plays on this row. Also supports HALT, LOOP, JUMP, and REM commands. Turn [LEVEL] or press [BANK]/[PTN] + [TRIG]
REP How many times this row repeats before moving to the next. Navigate to column, turn [LEVEL]
OF Pattern offset — which step the pattern starts playing from. Navigate to column, turn [LEVEL]
LN Override the pattern length for this row. Default uses the pattern's SCALE SETUP settings. Navigate to column, turn [LEVEL]
SCENE Two sub-columns: Scene A and Scene B assignment per row. Scenes come from the Part linked to that pattern. Turn [LEVEL] or press [SCENE A]/[SCENE B] + [TRIG]
T MIDI track transpose per row. A "T" appears when set. Press [YES] to open transpose window, turn [LEVEL]
B Per-row BPM override. A "B" appears when set. Overrides pattern and project tempo. Press [YES] to open tempo window, turn [LEVEL]
M Per-row track mutes. An "M" appears when set. [TRIG 1]-[TRIG 8] = audio tracks, [TRIG 9]-[TRIG 16] = MIDI tracks. Press [YES] to open mute window, press [TRIG] keys

Row Management

  • Insert a row: Press [FUNC] + [DOWN] to add a new row below the current position.
  • Delete a row: Press [FUNC] + [UP] to remove the currently selected row.
  • Copy/Paste/Clear: Standard [FUNC] + [REC] (copy), [FUNC] + [STOP] (paste), [FUNC] + [PLAY] (clear) operations work in the editor.
[FUNC] + [DOWN] → Insert row
[FUNC] + [UP] → Delete row

Position pointers: Two filled arrows on each side of a row show which row is currently playing. If you select a different row while playing, hollow arrows show which row will play next (once the current row finishes).

Jump to row: While the arrangement is playing, navigate to any row with [UP]/[DOWN] and press [YES]. The arrangement will jump to that row once the current row finishes — letting you skip ahead or go back on the fly.

Transitions & Repeats

The PAT column in the editor does more than just assign patterns. It also houses four special commands that control arrangement flow: HALT, LOOP, JUMP, and REM.

Special Commands

To access these commands, navigate to the PAT column of any row and turn the [LEVEL] encoder counter-clockwise past pattern A01. The commands appear in this order:

Command What It Does How to Set
HALT: Stops the arrangement at this row. Use at the end of your song, or to create a deliberate stop point. The arrangement will not loop past a HALT. Turn [LEVEL] counter-clockwise past A01 until HALT: appears.
LOOP: Loops a section of the arrangement. By default, loops infinitely between row 000 and the LOOP row. You can set a specific number of loop cycles using the [LEVEL] encoder on the value after LOOP: . From HALT, move focus to the value field, turn [LEVEL] one step counter-clockwise. Set loop count or leave as infinity.
JUMP: Jumps to a specific row number. The arrangement continues from the target row. Use this to create complex structures like A-B-A-C forms. From HALT, move focus to the value field, turn [LEVEL] clockwise. Set the target row number.
REM: A remark / comment row. Adds text to the arrangement as a visual cue. Does not affect timing or playback. Useful for labeling sections (INTRO, CHORUS, DROP). From HALT, turn [LEVEL] one more step counter-clockwise. Press [RIGHT] then [YES] to open the naming window.

Repeat Counts

The REP column sets how many times a row plays before the arrangement moves to the next row. Move the focus to the REP column using [LEFT]/[RIGHT], then turn the [LEVEL] encoder to set the value.

While the arrangement plays, a countdown appears to the right of the pattern name showing how many repetitions remain.

Song structure with REP: Set your verse pattern to REP x4, your chorus to REP x2, and your bridge to REP x1. You have just built a full song form from three patterns.

Per-Row BPM

The B column lets you set a different tempo for each row. Navigate to it and press [YES] to open the ROW TEMPO window, then turn [LEVEL] to set the BPM. Turn [LEVEL] counter-clockwise to the first setting ( PATTERN ) to use the pattern's own tempo instead.

Priority: Arranger BPM overrides both pattern tempo and project tempo. If your patterns sound too fast or too slow during arrangement playback, check the B column first.

Nested Loops

LOOP commands can be nested. For example, you can create an inner loop that repeats 4 times within an outer loop that repeats indefinitely. This creates complex structures from simple pattern combinations.

The Cue Transition Trick

This is the famous "smooth transition" technique. It lets you crossfade seamlessly between patterns even when they have completely different content. The idea: record the current mix into a buffer, crossfade to the recording, switch patterns behind the recording, then crossfade back to the live pattern.

Prerequisite: You should be comfortable with resampling basics from Tutorial 10 — Resampling .

Setup (One-Time)

1

Dedicate a track for the transition

Choose a track (e.g., [T8]) and assign a Flex machine to it. This track will hold the recording buffer.

[T8] → assign Flex machine
2

Set up the recorder

Press [T8] then open Recording Setup. Set SRC3 = MAIN to capture the main stereo output. Set SRC1 and SRC2 to - (no external input).

[T8] → [SRC] → Recording Setup → SRC3 = MAIN
3

Set recording length

Set RLEN (recording length) to match 1 or 2 bars at your tempo.

4

Place a one-shot recorder trig

In [REC] mode on [T8], place a recorder trig on step 1 by pressing [FUNC] + [TRIG 1]. This makes the recorder fire once and not re-record on subsequent loops.

[REC] → [FUNC] + [TRIG 1]

Performing the Transition

5

Arm the recorder

Press [TRACK] + [YES] on [T8] to arm recording. The recorder will capture the main mix on the next loop.

[TRACK] + [YES]
6

Crossfade to the recording

Set up your scenes so that [SCENE A] plays all live tracks and [SCENE B] solos the recording buffer on [T8]. Slide the crossfader toward [SCENE B]. You now hear the recording while the live tracks fade out.

7

Switch patterns

While the recording plays, switch to your next pattern. Press [PTN] + the target [TRIG] key (or select via [BANK] for a different bank). The audience hears a smooth sustain from the recording buffer while the new pattern loads.

[PTN] + [TRIG] → new pattern queues
8

Crossfade back to live

Once the new pattern is playing, slide the crossfader back to [SCENE A]. The new pattern fades in smoothly. No gap, no click.

Remember: Recorder buffers are volatile. They disappear on power off. If you capture something you love during a transition, save it to the CF card from the Audio Editor ([AED]) before shutting down.

Mutes & Live Control

Even with a pre-programmed arrangement, you still have full control over what the audience hears. Mutes, scenes, and the crossfader all work during arrangement playback.

Quick Mute During Playback

Press [FUNC] + [TRACK] to mute or unmute any track on the fly. The mute takes effect immediately. This works identically whether you are in normal pattern mode or arrangement mode.

[FUNC] + [T1] → Toggle mute on Track 1

Mixer Mutes

Alternatively, press [MIX] to open the Mixer menu, then use the [TRIG] keys to mute/unmute tracks: [TRIG 1] through [TRIG 8] control audio tracks, [TRIG 9] through [TRIG 16] control MIDI tracks.

[MIX] → [TRIG 1] — [TRIG 8] → mute audio tracks

Held mutes: In the Mixer, hold [FUNC] while pressing [TRIG] keys and the mute changes will be held until you release [FUNC]. This lets you set up multiple mute changes and release them all at once for a dramatic drop or build.

Per-Row Mutes (Automated)

In the Arranger editor, the M column lets you pre-program mutes for each row. Navigate to the M column, press [YES], then press the [TRIG] keys to toggle mutes. Rows with mute settings show an "M" indicator.

This means you can automate a drop — set row 3 to mute [T1] through [T6] so only drums and bass play, then row 4 unmutes everything for the chorus.

Scenes & Crossfader During Arrangement

The Arranger's SCENE columns assign [SCENE A] and [SCENE B] per row. But you can still override with the crossfader in real time. The arrangement sets the scene assignments , and you control how much of each scene is active via the crossfader position.

Combined with per-row tempo changes and mutes, you have full control over dynamics without touching a single knob during playback — or you can override everything live.

Parts and the Arranger

Each pattern is linked to a Part. When the arrangement switches patterns, the Part changes too — which means all machine assignments, effects, and scene settings change . Plan your Parts carefully so transitions sound musical.

Remember from Tutorial 08: Patterns that share a Part also share machine assignments. If your arrangement alternates between two patterns on the same Part, changing a machine on one changes it on the other. Use different Parts for patterns that need independent setups.

Practical Exercise

Build a 2-minute arrangement using 3 patterns. This exercise walks you through the entire process from start to playback.

Goal: Build a Verse-Chorus-Outro Arrangement

Phase 1: Prepare Your Patterns

You need at least 3 patterns in the same bank. If you have been following along from earlier tutorials, use patterns you have already created. Otherwise, create simple variations:

  1. Pattern A01 (Verse) — A basic beat. Kick on [TRIG 1], [TRIG 5], [TRIG 9], [TRIG 13]. Hi-hat on every other step.
  2. Pattern A02 (Chorus) — Same kick pattern but add a snare on [TRIG 5] and [TRIG 13]. Open the filter on [FX1]. Add more samples.
  3. Pattern A03 (Outro) — Strip it back. Just the kick and a long reverb tail on [FX2]. Maybe add a 50% condition on some trigs for fade-out variation.

Phase 2: Build the Arrangement

1

Enter Arrangement mode

Press [FUNC] + [ARR]. The [ARR] LED lights up.

[FUNC] + [ARR]
2

Open the editor

Press [ARR], select EDIT, press [YES].

[ARR] → EDIT → [YES]
3

Set Row 000: Verse (A01 x4)

Row 000 should already be selected. Navigate to the PAT column with [LEFT]/[RIGHT]. Set it to A01 using the [LEVEL] encoder or press [PTN] + [TRIG 1]. Move to the REP column and set it to x4 (plays 4 times).

[RIGHT] → PAT → [LEVEL] → A01 → [RIGHT] → REP → [LEVEL] → x4
4

Insert Row 001: Chorus (A02 x2)

Press [FUNC] + [DOWN] to insert a new row. Set PAT to A02 and REP to x2.

[FUNC] + [DOWN] → new row → set A02 x2
5

Insert Row 002: Verse again (A01 x4)

Press [FUNC] + [DOWN] again. Set PAT to A01, REP to x4. Notice how the Arranger lets you repeat the same pattern — unlike chaining.

6

Insert Row 003: Chorus again (A02 x2)

Press [FUNC] + [DOWN]. Set PAT to A02, REP to x2.

7

Insert Row 004: Outro (A03 x2)

Press [FUNC] + [DOWN]. Set PAT to A03, REP to x2.

8

Insert Row 005: HALT

Press [FUNC] + [DOWN] one more time. Navigate to the PAT column and turn [LEVEL] counter-clockwise past A01 until HALT: appears. This tells the arrangement to stop.

[LEVEL] → turn counter-clockwise → HALT:
9

Save the arrangement

Press [ARR] to return to the Arranger menu. Select SAVE and press [YES]. Name it (e.g., "SONG01") and press [YES] to confirm.

[ARR] → SAVE → [YES]
10

Play!

Press [PLAY]. The arrangement runs through: 4 bars verse, 2 bars chorus, 4 bars verse, 2 bars chorus, 2 bars outro, then stops at HALT. Use [FUNC] + [TRACK] keys to mute/unmute tracks as it plays.

[PLAY] → arrangement starts from row 000

Result: A complete verse-chorus-verse-chorus-outro song structure, fully automated.

Bars counter: While the arrangement plays, the header displays a counter showing how many bars and beats have played. This helps you track your position in the arrangement.

Arranger chaining: Once you have multiple arrangements saved, you can chain one arrangement to play after another. In the ARRANGER menu, select CHAIN, choose the next arrangement, and press [YES]. Only one arrangement can be chained at a time.
