What Is a Pickup Machine?

A Pickup machine turns the Octatrack into a loop pedal . Unlike Flex machines (which play pre-loaded samples from RAM) or Static machines (which stream from the card), a Pickup machine records live audio and immediately loops it . No pre-set recording length is required — the first recording defines the loop length.

The workflow is simple and physical:

  • Press [REC1] once to start recording
  • Press [REC2] to stop recording and start looping
  • Press [REC1] again to start overdubbing

This makes Pickup machines ideal for live performance : layering guitar loops, building vocal textures, stacking synth parts, or constructing entire compositions on the fly. Add a MIDI foot controller and you can do it all hands-free.

Key characteristics that make Pickup machines unique:

  • Hard-linked to track recorder and buffer — no manual buffer assignment needed
  • Cannot be sequenced or parameter-locked — they are live performance tools
  • Master/Slave system — the first recorded loop sets the tempo for everything
  • Use Flex RAM — they share the same 64 MB pool as Flex machines
  • Recordings are volatile — they disappear on power-off unless saved to the card

Manual reference: Section 9.3 — Pickup Machine Sampling (page 49). Appendix A.5 — Pickup Machine reference (page 121). Tutorial 17.1.4 (page 105).

Pickup Setup

Setting up a Pickup machine requires configuring the machine assignment, audio input source, monitoring, and fade settings. Follow these steps exactly.

Step-by-Step: Assign a Pickup Machine

1

Select a track

Press the [TRACK] key for the track you want to use (e.g. [T1] for Track 1).

[T1]
2

Open Playback Setup

Double-tap the [TRACK] key (or press [FUNC] + [SRC]) to open the PLAYBACK SETUP menu. The top of the screen shows the MACHINE parameter.

[T1] double-tap
3

Select Pickup machine

Turn the first encoder to scroll through machine types: FLEX, STATIC, THRU, NEIGHBOR, PICKUP. Select PICKUP . The track icon in the top bar changes to "P".

4

Open Recording Setup 1

Press [FUNC] + [REC1] to access RECORDING SETUP 1. This is where you configure the audio source.

[FUNC] + [REC1]
5

Configure the recording source

Set the parameters for your input:

  • INAB = A B (record from input pair AB)
  • INCD = - (disable CD inputs)
  • RLEN = MAX (allow maximum recording length)
  • TRIG = ONE2 (lets you manually define the loop end point)
  • SRC3 = - (no internal source — external only)
  • LOOP = ON
6

Open Recording Setup 2

Press [FUNC] + [REC2] to configure fades and monitoring.

[FUNC] + [REC2]
7

Set fade and monitoring

  • FIN = 0.063 (minimal fade in — prevents click at loop start)
  • FOUT = 0.063 (minimal fade out — prevents click at loop end)
  • AB = 127 (full monitoring of input pair AB)
  • QREC = OFF (no recording quantization for now)
  • QPL = OFF (no playback quantization for now)

Press [NO] to exit the menu.

[NO]

About FIN/FOUT: These add a very short fade to the recording to prevent clicks at the loop boundary. FOUT behaves differently for Pickup machines than for regular track recorders: the fade out is applied to the beginning of the captured loop (creating a crossfade at the loop point). Keep both at 0.063 for the cleanest transitions.

About TRIG modes: The TRIG setting in RECORDING SETUP 1 controls how the Pickup machine behaves. ONE uses the RLEN value to set a fixed loop length. ONE2 lets you press [REC1] or [REC2] to manually end the recording (most flexible). HOLD records only while you hold [REC1].

SRC Page Parameters

After assigning the Pickup machine, press [SRC] to see the PICKUP MAIN page. These parameters control playback behavior:

Parameter Function Recommended
PTCH Pitch of the loop (+/- 1 octave). Must be 0 for overdubbing/replacing to work. 0
DIR Playback direction: FWD, REV, or PIPO (ping-pong) FWD
LEN Slave loop length relative to master: OFF, X1, X2, X4, X8 (only affects slave machines) OFF (master) / X1 (slave)
GAIN Volume of recordings/overdubs in dB. Use negative values for gradual fade-outs. 0 dB
OP Operation mode: DUB (overdub on top), GAIN (volume changes only, no recording) DUB

Critical: If PTCH is set to anything other than 0, overdubbing and replacing will be disabled . Always return PTCH to 0 before attempting to record or overdub.

Master & Slave

The Pickup machine system has a master/slave hierarchy . Understanding it is essential for multi-track looping.

How Master Works

The first Pickup machine that records a loop automatically becomes the master . When the master loop is captured:

  • The Octatrack's BPM automatically changes to match the calculated tempo of the recorded loop
  • The master loop defines the base loop length that all slave loops sync to
  • The track icon shows "P" with two dots at the top-left = master

How Slaves Work

Any other Pickup machines on other tracks become slaves to the master. Slave behavior depends on the LEN parameter on the [SRC] page:

LEN Setting Slave Loop Length Use Case
X1 Same length as master Locked layers (bass, melody) that loop in perfect sync
X2 Double the master length B-section variations, longer melodic phrases
X4 4x the master length Extended progressions over 4 bars while master is 1 bar
X8 8x the master length Full song sections against a short master loop
OFF Free length (no sync to master) Independent loop — will drift against master

Slave track icons show "P" with two dots at the bottom-left .

Important: If all Pickup machines are stopped and a new one starts recording, that machine becomes the new master . Plan your track layout so the intended master always records first.

Sequencer Sync

You can lock the Octatrack's sequencer tempo to a Pickup machine loop. This is essential when mixing pre-programmed sequenced patterns with live Pickup loops:

[TRACK] + [TEMPO]

The BPM display will change to show which Pickup machine is controlling the tempo. If Pickup sync is active and the sequencer is stopped, pressing [PLAY] will start the sequencer in sync with the next loop restart.

Recording & Overdubbing

The heart of the Pickup machine is its loop pedal workflow. The function of the [REC1] and [REC2] keys changes when a Pickup machine is assigned to the active track. How they behave depends on the TRIG setting in RECORDING SETUP 1.

ONE2 Mode (Recommended)

ONE2 is the most flexible mode. It lets you manually define where the loop ends by pressing [REC1] or [REC2]:

State Action Result LED
Stopped Press [REC1] Recording starts Red (recording)
Recording Press [REC1] Stop recording, start overdubbing Orange (overdubbing)
Recording Press [REC2] Stop recording, start playback (no overdub) Green (playing)
Playing Press [REC1] Start overdubbing Orange (overdubbing)
Overdubbing Press [REC2] Stop overdubbing, continue playback Green (playing)
Playing Press [REC2] Stop playback Off (stopped)

ONE Mode

In ONE mode, the recording length is pre-set by the RLEN parameter. When [REC1] is pressed, the Pickup records for exactly that length and automatically enters overdubbing. Use [REC2] to switch to playback-only.

HOLD Mode

In HOLD mode, the Pickup records only while you hold [REC1] . Release [REC1] and it immediately starts looping. Press and hold [REC1] again to overdub. Replacing is not available in HOLD mode, but you can achieve the same effect by setting GAIN to -INF with OP set to DUB .

Overdub vs Replace

In ONE and ONE2 modes, you can toggle between overdubbing (layering on top) and replacing (overwriting the loop) using the [REC1] key while the loop is playing. Overdub = adds your new audio on top. Replace = erases the old audio and records fresh.

Saving Your Recordings

Pickup machine recordings live in the recorder buffer (Flex RAM). They are not saved automatically . If you power off the Octatrack, all Pickup recordings are lost. To save a recording permanently, open the RECORDING EDIT menu and save to the Compact Flash card:

[FUNC] + [REC3] → save recording to CF card

You can also save from the Audio Editor by pressing [AED].

Volatile recordings: Always save important Pickup recordings before powering off or changing projects. The recorder buffer contents are not retained across power cycles.

MIDI Foot Controller

A MIDI foot controller is the best way to use Pickup machines in a live performance. It frees your hands for playing an instrument while your feet control the loop recording.

Setup

  1. Connect the MIDI OUT of your foot controller to the MIDI IN of the Octatrack
  2. Make sure the foot controller sends on the Octatrack's auto channel (set in PROJECT > MIDI > CHANNELS)
  3. Assign Pickup machines to the tracks you want to control
  4. Configure the foot controller pedals to send MIDI notes

MIDI Note Mapping for Pickup Machines

The Octatrack responds to specific MIDI notes sent on the auto channel. These are the key mappings from Appendix C of the manual:

MIDI Note Note Name Function
60 C4 Combo Record — equivalent to [REC1]. Starts recording from all configured sources.
61 C#4 INAB Record — records from input pair AB only
62 D4 INCD Record — records from input pair CD only
63 D#4 SRC3 Record — records from internal source only
64 E4 Pickup Play/Stop — equivalent to [REC2]. Toggles playback.
68 G#4 Active Track Up — moves to next track
69 A4 Active Track Down — moves to previous track
71 B4 Toggle Seq Sync + Start — syncs sequencer to Pickup and starts

5-Pedal Setup Example

From the manual's tutorial (section 17.1.5): a practical 5-pedal foot controller layout for a guitarist:

Foot Controller Pedal Assignments

Pedal MIDI Note Function
Pedal 1 C4 (60) Start/stop recording (Combo Rec = [REC1])
Pedal 2 E4 (64) Play/Stop loop ([REC2])
Pedal 3 B4 (71) Sync sequencer to Pickup + start playing
Pedal 4 A4 (69) Previous track (move focus up)
Pedal 5 G#4 (68) Next track (move focus down)

CC alternative: If your foot controller only sends MIDI CCs (not notes), send CC 59 on the auto channel. The CC value is interpreted as the equivalent MIDI note number. For example, CC 59 with value 60 = Combo Record.

Workflow: Use Pedal 1 to start recording. Play your instrument. Hit Pedal 2 to end recording and start looping. Use Pedal 5 to switch to the next track. Hit Pedal 1 again to start a new layer on that track. Use Pedal 3 to sync the sequencer and mix Pickup loops with pre-programmed patterns.

RAM Considerations

Pickup machines use Flex RAM — the same 64 MB pool shared by all Flex machines and recorder buffers. Every second of Pickup recording eats into your Flex budget. Check your current usage by pressing [FUNC] + [PROJ] to open the PROJECT menu, then navigate to CONTROL > MEMORY. Plan carefully before committing to long loops.

RAM Usage per Second

Bit Depth Channels Per Second Per Minute
24-bit Stereo ~264 KB/s ~15.5 MB/min
16-bit Stereo ~176 KB/s ~10.3 MB/min
24-bit Mono ~132 KB/s ~7.7 MB/min
16-bit Mono ~88 KB/s ~5.2 MB/min

Practical Scenarios

At 120 BPM, a 4-bar loop = 8 seconds. Here is what that costs in RAM:

Scenario Duration RAM Cost (24-bit stereo) RAM Cost (16-bit mono)
1-bar loop @ 120 BPM 2 sec ~0.5 MB ~0.2 MB
4-bar loop @ 120 BPM 8 sec ~2.1 MB ~0.7 MB
8-bar loop @ 120 BPM 16 sec ~4.1 MB ~1.4 MB
16-bar loop @ 90 BPM ~43 sec ~11.1 MB ~3.7 MB

RAM savings tip: Record with mono input whenever possible (set INAB to A instead of A B ). A mono recording uses half the RAM of stereo. For guitar, bass, or single-mic vocals, mono is usually sufficient.

Shared pool: If you are also using Flex machines for drum samples or other loops, their RAM usage reduces what is available for Pickup recording. Press [PROJ] and navigate to CONTROL > MEMORY to check your RAM budget before starting a live looping session. Consider using [SRC] to switch some tracks to Static machines (which stream from the card at zero RAM cost) to free up Flex headroom.

Practical Exercise: Build a 3-Layer Loop

In this exercise you will build a multi-layer loop from scratch using three Pickup machines. You need an audio source connected to input A or AB (guitar, synth, mic, or a phone playing a backing track).

Goal

Create a 3-track loop with a rhythm layer, a bass/melody layer, and an ambient texture layer, each on its own track with independent effects.

Setup

1

Assign Pickup to Track 1 (Master)

Double-tap [T1] to open PLAYBACK SETUP. Set machine to PICKUP .

[T1] double-tap → select PICKUP
2

Configure Track 1 Recording

Press [FUNC] + [REC1]: set INAB = A B, INCD = -, RLEN = MAX, TRIG = ONE2, SRC3 = -, LOOP = ON.

Press [FUNC] + [REC2]: set FIN = 0.063, FOUT = 0.063, AB = 127.

[FUNC] + [REC1] → configure → [FUNC] + [REC2]
3

Configure Track 1 SRC page

Press [SRC] to open the PICKUP MAIN page. Set PTCH = 0, DIR = FWD, GAIN = 0, OP = DUB. Leave LEN at OFF (this is the master, LEN has no effect).

[SRC] → configure parameters
4

Assign Pickup to Track 2 (Slave)

Double-tap [T2], set machine to PICKUP . Copy the same RECORDING SETUP settings from Track 1. On the [SRC] page, set LEN = X1 (same length as master).

[T2] double-tap → PICKUP → [SRC] → LEN = X1
5

Assign Pickup to Track 3 (Slave)

Double-tap [T3], set machine to PICKUP . Same RECORDING SETUP. On [SRC] page, set LEN = X2 (double the master length for longer phrases).

[T3] double-tap → PICKUP → [SRC] → LEN = X2

Record the Loops

6

Record the Master Loop (Track 1)

Make sure [T1] is the active track. Play your instrument. Press [REC1] to start recording. Play a 4-bar rhythmic pattern. Press [REC2] to stop recording and start looping.

[T1] → play → [REC1] → perform 4 bars → [REC2]

The Octatrack's BPM will change to match your recorded loop. Track 1 is now the master .

7

Add effects to Track 1

Press [FX1] on Track 1 and add a filter or EQ. Press [FX2] and add a compressor or lo-fi effect. The effects process the loop in real-time.

[FX1] → choose effect → [FX2] → choose effect
8

Record Slave Loop on Track 2

Press [T2] to switch to Track 2 (the active track must be the one you want to record on). Press [REC1] to start recording. Play a bass or melody part. Press [REC2] to stop and loop. Because LEN = X1, this loop will be the exact same length as the master.

[T2] → [REC1] → perform → [REC2]
9

Record Slave Loop on Track 3

Press [T3] to switch. Record an ambient texture or pad. Because LEN = X2, this loop will be twice as long as the master — giving you an 8-bar phrase over the 4-bar rhythm.

[T3] → [REC1] → perform → [REC2]
10

Overdub and sculpt

Switch back to any track and press [REC1] to add overdubs. Use [FX1] and [FX2] on each track for different effects. Adjust track [LEVEL] to mix the layers. Try setting DIR = REV on Track 3's [SRC] page to reverse the ambient loop for a dreamy texture.

Performance tip: Use the [CUE] button + [TRACK] keys to preview individual layers in your headphones before unmuting them to the main output. This lets you rehearse an overdub before committing.

Remember to save: When you are happy with a loop, save it to the Compact Flash card via [FUNC] + [REC3] to open the RECORDING EDIT menu, then save. Otherwise it will be lost when you power off or change projects.
