SLICE & DICE
⏱ 30 MIN
⚡ BEGINNER/INT

1. WHAT ARE SLICES?

Slices are how you divide a sample into rhythmic chunks. Think of it like cutting a drum loop into individual hits. Each slice becomes its own playable segment, numbered 1-64 (max 64 slices per sample).

Once sliced, you can play any slice independently, chop them into a new rhythm, reverse them, pitch them, or sequence them. This is the foundation of creative loop chopping on the Octatrack.

KEY CONCEPT
A drum loop at 128 BPM might be 8 bars long. If you grid-slice it into 32 slices, you get roughly 2 slices per beat (32 slices ÷ 16 steps = 2 per beat). Now you can play those slices in any order, creating completely new breakbeats.

2. THE AUDIO EDITOR

The Audio Editor is where you view, slice, and edit samples. To access it:

MENU PATH
1. Select a track with a sample loaded
2. Press [FUNC] + [AUD] to open Audio Editor
3. You'll see the waveform displayed

Navigation in Audio Editor:

  • [LEFT]/[RIGHT] arrows - move playhead through sample
  • [FUNC] + [LEFT]/[RIGHT] - zoom in/out on waveform
  • [UP]/[DOWN] - adjust waveform zoom depth
  • [YES]/[NO] - confirm/cancel slicing actions

The waveform shows silence as flat lines and loud sections as tall peaks. You'll use these visual cues to find where to place slice markers.

3. GRID SLICING: THE INTERACTIVE WAY

Grid slicing divides a sample into EVENLY SPACED sections. Perfect for drum loops that are locked to a tempo. Try it below:

Select number of slices and click APPLY GRID. Each vertical line = one slice.
MENU PATH
Audio Editor > [FUNC] (to access menu) > Slice > CREATE SLICE GRID
Select number of divisions (4, 8, 16, 32, 64) > [YES] to confirm

When to use grid slicing:

  • OK Drum loops locked to tempo
  • OK 4/4 time signatures (most loops)
  • OK When you want even rhythmic divisions
  • OK Quick way to slice without thinking

4. MANUAL SLICING: PRECISION CHOPPING

For irregular samples or precise control, manual slicing lets you place markers exactly where you want. Here's the process:

STEP-BY-STEP
1. In Audio Editor, use [LEFT]/[RIGHT] to find your slice point
2. Press [YES] to add a marker at that position
3. The marker appears as a vertical line in the waveform
4. Continue placing markers throughout the sample
5. Press [NO] when done, then confirm to save slices

CRITICAL: Zero-Crossing Snap

Always enable ZERO-CROSSING SNAP before slicing. This automatically places markers at points where the waveform crosses zero (silence). Why? Prevents clicks and pops at slice boundaries when you play them back.

ZERO-CROSSING SNAP
Audio Editor menu > Snap > ZERO CROSSING
Enable this, then place markers. The OT will snap them to the nearest zero crossing automatically.

Manual slicing takes longer than grid slicing, but gives you surgical precision. Perfect for one-shots, vocal chops, or complex loops where grid divisions don't make sense.

5. PLAYING SLICES: TURNING CHUNKS INTO BEATS

Once your sample is sliced, you need to enable slice playback in the track's Playback Setup:

ENABLE SLICE PLAYBACK
[TRACK] button on the sliced track > Playback Setup > SLICE = ON

Now each [TRIG] key can play a specific slice. Here's the interactive demo below - you're building a chopped breakbeat from 16 slices. Click each step to assign a slice number:

SLICE SEQUENCER (16 STEPS)
HOW TO P-LOCK SLICES
1. In GRID RECORDING mode ([REC])
2. Hold [TRIG] key you want to lock
3. Turn the encoder to select slice number (1-64)
4. Release [TRIG]
5. That step now plays the selected slice

The sequencer above shows what a chopped beat looks like. Notice how steps are assigned to different slices? That's you rebuilding the rhythm from the original sample.

6. SLICE LOCK TECHNIQUES: BEYOND THE BASICS

Slices are just the start. You can manipulate each slice independently:

TECHNIQUE HOW WHY
Reverse slices P-lock RATE to negative (-100) Backwards hits, glitch effects
Pitch slices P-lock TUNE (semitones) Tuned drum kit, melodic loops
Repeat slices Place same slice number on consecutive steps Stutter effects, rhythmic emphasis
Skip slices Leave gaps in slice sequence Syncopation, variation, space
Randomize with LFO LFO targeting SLICE parameter Generative patterns, chaos
EXAMPLE: GLITCH BEAT
1. Slice a drum loop into 32 pieces
2. Sequence slices 1-16 normally
3. On step 9, reverse the slice (RATE = -100)
4. On step 13, pitch it up 7 semitones
5. On step 15, repeat step 14's slice twice
Result: a recognizable loop with glitchy variation

7. .OT FILES & SAMPLE CHAINS (ADVANCED)

When you slice a sample on the OT, it creates a .ot metadata file alongside the WAV. This file stores:

  • Slice point locations (exact sample numbers)
  • Original tempo and key
  • Gain and loop settings
  • Playback mode
SAMPLE CHAINS
A chain = multiple sounds concatenated into ONE WAV file + ONE .ot file. Each "sound" becomes a slice. So 64 samples become 64 slices in a single slot = massive space savings.

Why use chains?

  • OK 64 drum hits in 1 slot instead of 64 slots
  • OK Saves RAM (one Flex allocation)
  • OK Faster navigation (fewer files)
  • OK Keep related samples together

Build chains with:

  • OctaChainer (free Mac app, UI-based)
  • Python + ffmpeg (CLI, if you're scripting)

Best practice: organize chains by power-of-2 divisions (16, 32, 64 slices). Easier to reason about slice boundaries.

Slice & Dice — Chopping Loops Into Pieces

Slicing is the art of dividing a single sample into multiple playable segments. Whether you're chopping a drum loop into individual hits, rearranging a melodic phrase, or creating variations from a single source, slicing is one of the Octatrack's most powerful creative tools.

In this tutorial, you'll learn two slicing methods, build an interactive waveform editor, and discover how to use slices to transform loops into entirely new rhythms and melodies.

What Are Slices?

A slice is a marker in an audio file that defines a starting point for playback. Think of it like bookmarks in a long document—each bookmark lets you jump to a different part of the sample.

Key Facts

  • Max 64 slices per sample — This gives you up to 64 different playable segments
  • Each slice = a different starting point in the waveform
  • Works on Flex AND Static machines — Though Flex is more responsive
  • Slice numbers are p-lockable — You can change which slice plays on each step

When to Use Slices

  • Chop drum loops → Isolate individual kicks, snares, hats
  • Rearrange melodies → Play notes in a different order
  • Create variations → Play the same loop backwards, stutter, skip parts
  • Glitch & experiment → Randomize slice order, combine slices with pitch/filter changes

Grid Slicing — Automatic, Evenly-Spaced Markers

The fastest way to slice. The Octatrack divides your sample into evenly-spaced segments—perfect for drum loops with consistent timing.

Step-By-Step: Grid Slicing

  1. Load a sample on a Flex track — Navigate to a track's machine, assign Flex, load a loop
  2. Open Audio Editor — Press [FUNC] + [AUD EDIT] (or [FUNC] + [BANK] on older software)
  3. Navigate to the Slice page — Use [UP]/[DOWN] to find "Slice"
  4. Select CREATE SLICE GRID — Press [YES] to confirm
  5. Choose divisions — Use the encoder to pick: 4, 8, 16, 32, or 64 slices
  6. Confirm — Press [YES]. The OT places markers evenly across the sample
  7. Return to main page — Press [FUNC] to close Audio Editor

Best For

  • Drum loops with a clear, steady tempo
  • Any loop that feels "in time" — BPM-locked samples
  • Quick setup — seconds instead of minutes

Pro Tip

If your loop is 8 bars long, choose 32 slices to get one slice per beat. If it's 4 bars, use 16 slices. This makes rearranging easier because each slice represents a musical unit (1 beat).

Manual Slicing — Precise, Step-By-Step Placement

For irregular rhythms, vocal chops, or specific hit isolation, manually place each slice marker exactly where you need it.

Step-By-Step: Manual Slicing

  1. Open Audio Editor — Press [FUNC] + [AUD EDIT]
  2. Navigate to Slice page — Use [UP]/[DOWN] arrows
  3. Position the playhead — Use the big encoder to scrub through the waveform
  4. Listen (optional) — Hold [YES] to preview the audio at the current position
  5. Add a slice marker — Press [YES] when the playhead is at your desired slice point
  6. Repeat — Move to the next slice point and press [YES] again
  7. Enable ZERO-CROSSING SNAP — Press [FUNC] + [PARAMETER] to access Slice settings → turn on SNAP for clicks-free slicing

Best For

  • Vocal samples — isolate individual words or syllables
  • Synth stabs — separate chords
  • Irregular rhythms — samples that don't fit a grid
  • Micro-precision — when you need exact control

ZERO-CROSSING SNAP (Critical!)

Always enable this. It snaps your slice markers to zero-crossing points in the waveform—places where the audio wave crosses silence. This prevents audible clicks when the OT jumps between slices.

Playing Slices — Making Them Work in Your Pattern

Once you've created slices, you need to tell the Octatrack which slice to play on each step. This is where slicing becomes powerful.

Setup: Enable Slice Playback

  1. Select a sliced sample on a track
  2. Open Playback Setup — [FUNC] + [PLAYBACK]
  3. Find the SLICE parameter — Navigate with [UP]/[DOWN]
  4. Set SLICE = ON — Now the track will respond to slice commands
  5. Exit — Press [FUNC] to return to pattern

Playing Different Slices Per Step

Now, each step can trigger a different slice:

  • Step 1, Slice 1 — first part of the loop
  • Step 2, Slice 5 — jump to slice 5
  • Step 3, Slice 2 — back to slice 2
  • Step 4, Slice 1 — repeat slice 1

This rearrangement lets you create entirely new rhythms and melodies from the original sample.

Slice Locks Per Step — The Grid Recording Technique

Just like sample locks let you play different samples on each step, slice locks let you play different slices on each step.

Step-By-Step: Using Slice P-Locks

  1. Enter GRID RECORDING mode — Press [REC]
  2. Hold a [TRIG] key — e.g., step 1
  3. Turn the SLICE encoder — While holding the trig, adjust the SLICE parameter with the big encoder
  4. Release trig — That step now has a slice lock
  5. Repeat — Move to step 2, hold [TRIG], adjust SLICE, release
  6. Preview — Press [PLAY]. Each step plays a different slice

Visual Feedback

When a step has a slice lock, the trig LED flashes RED. This tells you at a glance which steps have custom slice assignments.

Example: Drum Loop Rearrangement

Original loop: Kick, Snare, Hat, Clap (repeating)

Sliced to 4 parts. Now rearrange:

  • Step 1: Slice 1 (Kick)
  • Step 2: Slice 2 (Snare)
  • Step 3: Slice 4 (Clap) ← Swapped!
  • Step 4: Slice 1 (Kick) ← Repeated

Result: A completely new drum pattern from the same original loop.

Creative Slicing Techniques

1. Glitchy Melodies — Randomize Slice Order

Slice a melodic loop into 8 or 16 slices. Manually p-lock them in random order. The result: a chaotic, glitchy reharmonization of the original melody.

2. Vocal Chops — Isolate Words & Syllables

Record a vocal phrase. Manually slice at each word boundary. Now trigger different words in a new order to create new lyrics or rhythmic stutters.

3. Pitch-Shifted Slices — Melodic Instrument

Slice any sample. On the same track:

  • Slice 1: Original pitch (0 semitones)
  • Slice 2: +2 semitones
  • Slice 3: +4 semitones
  • Slice 4: +7 semitones

Now p-lock both SLICE and PITCH per step. You've turned one sample into a chromatic instrument.

4. Auto-Slice Cycling with LFO

Assign an LFO to the SLICE parameter. Set it to cycle through slices automatically. Adjust LFO speed for rhythmic chopping, stuttering, or smooth slice transitions.

5. Combining Slices with TSTR (Timestretch)

Each slice can have its own TSTR setting. Slice a loop, then on some steps use TSTR to speed up, on others slow down. Creates rhythmic variety while staying locked to the grid.

Sample Chains (.ot Files) — 64 Sounds in 1 Slot

A sample chain is multiple separate audio files stitched together into a single WAV, with metadata (.ot file) that tells the Octatrack where the slice boundaries are.

Why Use Chains?

  • Save slots: 64 different sounds in 1 Flex slot instead of 64 slots
  • Organize: Keep related samples (drum kit, synth stabs, vocal chops) together
  • Convenient: Load a full kit with one file browser navigation

How to Build a Sample Chain

Option 1: OctaChainer (Free macOS App)

  1. Download OctaChainer from the community
  2. Load 16 or 32 audio files (WAV, AIFF, 44.1kHz)
  3. Set slice count (should match file count, or leave auto)
  4. Export as .wav + .ot file
  5. Copy both files to your AUDIO/ folder

Option 2: Claude Code (With Python & FFmpeg)

Claude can automate chain building with ffmpeg + Python scripting. If you have samples ready, ask Claude to concatenate them and generate the .ot metadata file.

Best Practices for Chains

  • Use 16 or 32 slices (powers of 2) for easiest mental math
  • All files should be 44.1kHz, 16 or 24-bit, mono or stereo
  • Keep individual files in the chain roughly equal length (1-2 seconds each)
  • Name the chain clearly: Drumkit_Boom_02.wav
